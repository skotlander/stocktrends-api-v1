# STOK Token Strategy (API Integration)

## Objective

Integrate STOK into the API without breaking pricing fairness.

---

## Principles

* STC remains the base pricing unit
* STOK does NOT replace STC
* STOK modifies access economics

---

## Use Cases

### 1. Discounts

* STOK holders receive reduced STC cost

---

### 2. Access Tiers

* premium endpoints unlocked

---

### 3. Game Integration

* API usage linked to game participation
* rewards tied to STOK ecosystem

---

## Risk

* inflation of STOK supply
* unfair advantage if not balanced

---

## Design Constraint

STOK must NOT distort core pricing fairness

---

## Outcome

Token-enhanced API economy

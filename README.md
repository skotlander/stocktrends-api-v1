# Stock Trends API

FastAPI service providing programmatic access to Stock Trends market data.

## Local Development

Install dependencies:

pip install -r requirements.txt

Run API:

run_api.bat

Docs:

http://127.0.0.1:8000/docs

## API Version

v1

## Major Endpoint Groups

/instruments
/prices
/indicators
/stim
/selections
/stwr
/breadth
/leadership
